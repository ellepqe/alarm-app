# Alarm App - Version History

## v1.0.0 - Initial Release (2026-05-27)

### Features
- ✅ Full alarm management system
- ✅ Sun and Moon time tracking
- ✅ Note-taking functionality
- ✅ Multi-language support
- ✅ Data export (JSON & Text)
- ✅ Notification system
- ✅ Alarm scheduling with AlarmManager
- ✅ Vibration and sound support

### Technical Stack
- Kotlin with Coroutines
- MVVM Architecture
- Room Database
- Android Jetpack Libraries
- Material Design 3
- GitHub Actions CI/CD
