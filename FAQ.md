# Frequently Asked Questions

## Installation

### Q: Why does my phone say "Install from Unknown Source"?
A: Android requires permission for apps not from Play Store. Go to Settings → Security → Unknown Sources and enable it.

### Q: Can I uninstall a previous version?
A: Yes! Uninstall the old app first, then install the new APK. Your alarms data will be preserved if you have backups.

### Q: How much storage does the app need?
A: The APK is about 5-10MB. The app uses minimal storage for alarm data.

## Features

### Q: Will alarms work if my phone is off?
A: Alarms will trigger when the phone is on. If it's off during alarm time, it will trigger when powered on (if enabled).

### Q: Can I set multiple alarms for the same time?
A: Yes! You can create unlimited alarms with different names and settings.

### Q: Do sun/moon times update automatically?
A: Yes, they update based on your device's location and time settings.

### Q: Can I export my alarms?
A: Yes! Go to Settings → Export JSON/Text to save your data.

## Troubleshooting

### Q: App keeps crashing
A: Try these steps:
1. Clear app cache: Settings → Apps → Alarm App → Clear Cache
2. Uninstall and reinstall
3. Ensure Android 5.0+ is installed
4. Check for sufficient storage space

### Q: Alarms not triggering
A: Check:
1. Alarm is enabled (toggle switch ON)
2. Correct time is set
3. Volume is not muted
4. Battery optimization is not blocking the app
5. System time is correct

### Q: Where is my data stored?
A: All data is stored locally on your phone in the app's private database. No data is sent to servers.

### Q: Can I sync data between devices?
A: Currently, no cloud sync. Use the Export/Import feature to manually transfer data.

## Updates

### Q: How do I update the app?
A: 
1. Download the new APK from [Actions](https://github.com/ellepqe/alarm-app/actions)
2. Install it (replaces the old version)
3. Your data is preserved

### Q: Are updates automatic?
A: Not yet. You need to manually download and install new APKs.
